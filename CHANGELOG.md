# Changelog

All notable changes to this project are documented in this file.

## 0.1.0 - 2026-03-09

- Bootstrapped the standalone Adaptive TV Remote Spring Boot + PWA service
- Added auto-routing across LAN, IR gateway, and HDMI-CEC gateway paths
- Added brand adapter skeletons for Samsung, LG, and Sony LAN control
- Added real-integration skeletons for LAN brands and gateway IR/HDMI-CEC clients
- Added standalone repository metadata, CI scaffolding, and contributor docs

## 0.1.1 - 2026-03-11

- Added production quick start and real/mock profile configs
- Implemented Sony IRCC command dispatch and handshake validation
- Added gateway payload formats for IR/CEC plus TCL IR templates
- Added TCL demo device/candidate and onboarding flow updates
- Expanded mock integration tests with CI tagging and reporting
- Introduced structured integration error codes
